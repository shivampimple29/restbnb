# 🏡 RestBNB – Full-Stack Vacation Rental Platform

A production-ready **Airbnb-inspired rental platform** built using **Node.js, Express, MongoDB, and EJS**, focused on real-world backend architecture and cloud deployment.

🌐 **Live Website:**  
👉 https://restbnb-e1xk.onrender.com

---

## 📌 Project Overview

Modern rental platforms require more than just a user interface — they need scalable backend architecture, smart data handling, authentication, and cloud deployment.

**RestBNB** is a full-stack vacation rental application that simulates a real-world property booking platform.  
It demonstrates how backend systems connect to cloud databases, manage users securely, calculate dynamic property scores, and operate smoothly in production.

---

## 🎯 Objective

To design and deploy a production-ready rental platform that:

- Performs complete CRUD operations  
- Connects to a cloud-hosted MongoDB database  
- Implements smart search and filtering  
- Calculates dynamic property scores  
- Integrates interactive maps  
- Handles authentication securely  
- Runs successfully in production (Render)  

---

## 🚀 Key Features

- 🏘️ Create, Edit, and Delete Listings  
- 🔎 Smart Keyword Search  
- 🎯 Dynamic Category Filters (Mountains, Cities, Camping, etc.)  
- 📊 Property Score System (Host, WFH, Area Safety)  
- 🏷️ Dynamic Badge System (WiFi, Power Backup, Water 24x7)  
- 🗺️ Interactive Mapbox Location Display  
- 🔐 User Authentication (Login / Signup / Logout)  
- 💬 Review and Rating System  
- 🔔 Flash Success and Error Messages  
- 📱 Fully Responsive UI  
- ☁️ Cloud-Hosted Database

---

## 🧩 Application Flow

1. User visits the deployed website  
2. Server runs on a dynamic production port  
3. Application connects to cloud MongoDB  
4. Listings and reviews are fetched dynamically  
5. Search and filters refine displayed results  
6. Property scores are calculated server-side  
7. Maps display listing locations  
8. Authentication protects restricted actions
